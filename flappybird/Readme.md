# FlappyBird terminal game 
Game for deep learning and neural network to play
---
## How to Run วิธีรันเกม
1. ใช้ Compiler C++ เช่น MinGW หรือ Visual Studio
2. เปิด Terminal แล้ว compile
---
## Controls ปุ่มกด
- [SPACE] กระพือปีก 
- [Q] กดออกเกม
- [SPACE] อีกรอบเพื่อรีเกมใหม่เวลาแพ้
